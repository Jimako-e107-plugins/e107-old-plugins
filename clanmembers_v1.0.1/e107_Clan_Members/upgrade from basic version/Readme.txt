UPGRADE TO FULL VERSION

- First upload the content of the hmtl folder to your root directory
- CHMOD the all folders in the e107_plugins/clanmembers/images folder to 777
- Upload Upgrade.php to your root directory
- Run the script by going to http://www.yoursite.com/Upgrade.php
- The plugin is now upgraded to the full version

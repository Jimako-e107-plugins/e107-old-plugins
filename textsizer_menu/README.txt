############## "Textsizer" plugin ####################### 

Purpose:

  This is a plugin for the E107 CMS system (e107.org).
  This plugin allows users of your site to increase and decrease text size.
    
Requirements:

  This plugin requires e107 Verion 0.700+

############## INSTALLATION ####################### 

1) Upload all files to your e107_plugins\ directory on your server, retaining directory structure.
2) Go to the admin section of the website, go the to plugin manager and install the Textsizer.
3) Add {TEXTSIZER} shortcode to your theme.php or news template.

############## "Textsizer" модул ####################### 

Цел:

  Този модул е за E107 CMS системи (e107.org).
  Този модул позволява на потребителите на сайта ви да увеличават и намаляват размера на текста.
    
Изисквания:

  Този модул изисква e107 Версия 0.700+

############## ИНСТАЛАЦИЯ ####################### 

1) Разархивирайте и качете папката с файловете във вашата e107_plugins\ директория на сайта ви.
2) Вървете в админ секцията на сайта ви, кликнете на Модул Мениджър, намерете модула "Textsizer" и го инсталирайте.
3) Добавете {TEXTSIZER} кода във файла theme.php на темата която ползвате или в news template.

Polski
############## Plugin "Textsizer" ####################### 

Przeznaczenie:

  Plugin stworzony został dla systemu CMS - E107 (e107.org).
  Plugin umożliwia użytkownikom odwiedzającym twoją stronę zmniejszać lub powiększać rozmiar tekstu.
    
Wymagania:

  Plugin przeznaczony jest dla e107 w wersji 0.700+

############## INSTALACJA ####################### 

1) Skopiuj wszystkie pliki do katalogu e107_plugins\ znajdującym się na serwerze, zachowując strukturę katalogów.
2) Przejdź do panelu zarządzania twojej strony, znajdź w menedżerze pluginów Textsizer. Zainstaluj go. 
3) Aktywuj menu textsizer lub dodaj shortcode {TEXTSIZER} w odpowiednim miejscu pliku theme.php lub szablonu aktualności.

-------------------------------------------------------------------------------------------------------------+

  INSTALL
  
  - Unzip, and upload donate_menu folder to /e107_plugins/
  
    ( folder structure should look like /e107_plugins/donate_menu/plugin.php )

  - Admin Area, Plugin Manager, Install
  
  - Admin Area, Donate Plugin, Save Your Settings
  
  - Admin Area, Menus, Activate the Donate Menu
  
-------------------------------------------------------------------------------------------------------------+

  UPGRADING
  
  - Backup any custom donation button images
  
  - Delete the donate_menu folder ( This is to remove files no longer needed )
  
  - Upload New version
  
  - Admin Area, Plugin Manager, Upgrade
  
  - Admin Area, Donate Plugin, Check and Re-save the settings

PView Gallery v1.3.2B and later versions

deutsch: (english below)

1. Erstelle ein Backup aller Dateien.
2. Lösche alle Templates. Wichtig!
3. Kopiere alle Dateien (alle überschreiben)
4. Benutze den Pluginmanager des e107, um das Datenbankupdate durchzuführen.
5. Stelle im Adminbereich die Ansichten und die Startseite ein.
6. Profilseiten Erweiterung ist im Adminbereich beschrieben: Shortcode im theme template einfügen


english:

1. Do a backup of all files.
2. Delete all templates. Important!
3. Copy all files, overwrite all
4. Use pluginmanager of e107, to update database
5. Adjust the views and frontpage by using adminpanel
6. The profile page extension is described in adminpanel: insert shortcode in theme template

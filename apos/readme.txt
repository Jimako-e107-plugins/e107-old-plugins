Installation -
	1) Extract and Upload to e107_plugins directory
	2) Cofigure plugin
	
Config
	1) Select where to post thread
	2) Enter the USER ID number who the post will be from
	3) Enter Subject and Thread text

Tips
* The user id can be found in Admin > Users
* For custom users, just Quick Add User in the admin area and use the new user id
* Not tested on Admin Approval verification method
